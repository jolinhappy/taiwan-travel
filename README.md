# 【前端修練精神時光屋】 - Taiwan Travel
使用TDX運輸資料流通服務，以及UI設計師投稿的設計製作的台灣旅遊景點導覽。


## 網站功能
- 使用者可依照類別及地區搜尋景點。
- 使用者可依照類別及地區搜尋活動。
- 使用者可查看景點或活動的詳細資訊。

## 環境建置與需求
- axios: v0.24.0
- vue: v3.0.0
- vue-router: v4.0.0
- element-plus: v1.2.0-beta.2
- node-sass: "4.12.0"
- sass-loader: "8.0.2",
